# ferrum-core-macros

Procedural macros for [ferrum-core](https://crates.io/crates/ferrum-core).

## Macros

- `#[derive(FerrumRecord)]` — derive structured record types for array metadata
- `s![]` — NumPy-style slice macro for array indexing
- `promoted_type!` — compile-time type promotion resolution

## Usage

This crate is a dependency of `ferrum-core` and is not intended for direct use. Depend on [`ferrum`](https://crates.io/crates/ferrum) or [`ferrum-core`](https://crates.io/crates/ferrum-core) instead.
